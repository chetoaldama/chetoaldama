## 🏗️ Flagship Projects

### 🔥 Puffer — *Coming Soon to Open Source*
An ambitious multi-component system with a **core engine**, **Chrome extension**, and **dedicated server**. Built for scale, automation, and real-time performance. *(Core, Chrome Extension, Server — three repos)*

### 🩺 Psicovital
A complete **telemedicine ecosystem**: mobile app (Expo/React Native), web dashboard, REST API, admin panel, real-time chat, cron scheduling, and backend services. Connecting patients with mental health professionals digitally.

### 🤖 Sugos AI
Next-gen **AI agent platform** integrating Large Language Models, MCP (Model Context Protocol), and ElevenLabs voice synthesis — bringing conversational AI to life with natural, expressive interactions.

### 📊 Smmartt — *Intelligence Commerce Platform*
A sophisticated **business intelligence & AI extraction** platform. Analyzes data, extracts actionable insights, and powers smarter commerce decisions.

### 📈 Trading Terminal & Bot
A **real-time trading system** built with Next.js (frontend terminal) + Python (automated trading bot). Market data, strategy execution, and portfolio management in one cohesive system.

### 🛵 TuCarritoFoodsV2
Food delivery platform — handling orders, riders, restaurants, and real-time logistics. Built for the LATAM market.

### 🗂️ OneData / OneDate
Full-stack **data management platform** with PostgreSQL backend, RESTful API, and modern frontend. Education, backups, structured data at scale.

### 🍃 Visite Mérida
AI-powered tourism platform for the Mérida region — because Venezuela's beauty deserves a world-class digital experience.
