## 🏗️ Projects I've Built

> **3,383+ commits • 43+ repos • 10+ clients** — this is what I've shipped.

### 🔥 Puffer — *Multi-Component System (My Flagship)*
A large-scale system built across **three repositories** (Core, Chrome Extension, Server). Monorepo architecture with apps, packages, tests, benchmarks, and CI/CD. Built with TypeScript, Node.js, and modern tooling. The kind of project that demands real architectural thinking.

### 🩺 Psicovital — *Complete Telemedicine Ecosystem*
The biggest project I've built: **6 interconnected repos** spanning a React Native mobile app, web dashboard, REST API, admin panel, real-time chat, cron jobs, and backend services. **1,890+ commits** across all components. Connects patients with mental health professionals digitally — appointments, payments, chat, medical records, the whole stack.

### 🤖 Sugos AI — *Multi-Agent Voice Platform*
An AI agent platform integrating **Large Language Models + MCP (Model Context Protocol) + ElevenLabs voice synthesis**. Multiple components: agent backend, Expo mobile app, MCP server, ElevenLabs integration, and a web frontend. Conversational AI with real-time voice, tool calling, and agent orchestration.

### 📊 IA Extract / Smmartt — *Commerce Intelligence*
A **business intelligence & AI extraction platform** — 368+ commits of data ingestion, entity extraction, and actionable analytics. Smmartt extends this into a full commerce intelligence ecosystem. Built for real business decisions, not demos.

### 📈 Trading Terminal + Bot — *Real-Time Markets*
A **Next.js 16 trading terminal** with a **Python FastAPI automated trading bot**. Real-time market data, strategy execution, portfolio management, and backtesting. Dockerized, production-ready. *Because money moves fast.*

### 🧠 dontworrydude — *AI Commercial Intelligence Platform*
A **TypeScript monorepo** (pnpm workspaces) combining backend + frontend + AI. Prisma ORM, Docker deployment, CI/CD pipelines. AI-powered commercial intelligence for real-world business analysis.

### 🔐 Proton — *Voting System with Hash Validation*
A **Cloudflare Workers + frontend** voting system with cryptographic hash validation, anti-fraud device checks, and real-time results. Edge-deployed, serverless, tamper-resistant architecture. Security-first design.

### 💰 Administración de Divisas — *Currency Management*
A full currency administration and remittance platform for the Venezuelan market. Manages exchange rates, transactions, and compliance in a complex regulatory environment.

### 🏪 TuCarritoFoodsV2 — *Food Delivery Platform*
Orders, riders, restaurants, real-time logistics — built for the LATAM food delivery market. GPS tracking, payment processing, order state management.

### 🗂️ OneData / OneDate — *Data Management Platform*
Full-stack PostgreSQL + REST API + React frontend. Data management at scale with structured schemas, backups, and an education module.
